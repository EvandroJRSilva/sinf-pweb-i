Entendendo o CSS Box Model (2026)

Regra moderna recomendada: * { box-sizing: border-box; }

Isso faz com que width e height representem o tamanho final visível (content + padding + border).

Sem isso, adicionar padding ou border "estoura" o layout.

content-box (padrão antigo)
width: 300px declarado
→ Tamanho real na tela: 300 + 30×2 + 15×2 = 390px
Observe como a caixa vermelha ficou maior que os 300px declarados
border-box (recomendado hoje)
width: 300px declarado
→ Tamanho real na tela: exatamente 300px
(o conteúdo encolhe automaticamente)
Aqui o layout fica previsível e intuitivo
Margem superior: 60px
Margem lateral: 40px (shorthand)
Margem superior declarada: 80px
Mas colapsou com a de cima → distância real entre caixas = 80px (não 140px)
